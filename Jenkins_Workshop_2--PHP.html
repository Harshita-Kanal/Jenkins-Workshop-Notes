PHP

- Install PHPUnit & JUnit plugins

(JUnit required to parse the results bu PHPUnit test plugin which saves the output in unit_test_results.xml format)

Repo: https://github.com/miteshgoplani/jenkins-phpunit-test

Command: phpunit tests/GumballMachineTest.php

On Jenkins Server: sudo apt-get install phpunit


Add Build Step -> Execute Shell
phpunit --log-junit results/phpunit.xml -c tests/phpunit.xml


Post Build Actions -> Publish JUnit Test results report
results/phpunit.xml
