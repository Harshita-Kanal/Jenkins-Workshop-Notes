Configuring NodeJS

Manage Jenkins > Global Tool Configuration > NodeJS





Select Node Version and Add any global packages you need



Project Type: Pipeline



Pipeline Example

pipeline {
agent any

tools {nodejs "node"}

stages {
stage('Example') {
steps {
sh 'npm config ls'
}
}
}
}


Explanation

1. agent any
• The job will run in any jenkins agent.
• You can have a lot of jenkins nodes, one master and some slaves, and the job will run in any node.

2. tools {nodejs "node"}
• This line search for nodejs tool named “node” and use it in pipeline.

3. stages {
• Stages is the inicialization from pipeline steps.

4. stage('Example') {
steps {
...
}
}

• This commands start a new stage named “Example” and run the steps inside this stage.

5. sh 'npm config ls'
• It execute a sh script in machine. With this, we can test the npm running correctly.

Clone and Run Test

pipeline {
agent any

tools {nodejs "node"}

stages {

stage('Cloning Git') {
steps {
git 'https://github.com/miteshgoplani/jenkinsNodeTest.git'
}
}

stage('Install dependencies') {
steps {
sh 'npm install'
}
}

stage('Test') {
steps {
sh 'npm test'
}
}
}
}
